Passos para enviar para o GitHub:

            
# 1. Abra o terminal e navegue até a pasta do seu site:
cd caminho/para/a/pasta/meu-site

# 2. Adicione o repositório remoto do GitHub:
git remote add origin https://github.com/SEU_USUARIO/meu-site.git

# 3. Adicione os arquivos ao controle de versão:
git add .

# 4. Faça o commit com uma mensagem:
git commit -m "Adiciona arquivo index.html"

# 5. Renomeie a branch principal para "main" (se necessário):
git branch -M main

# 6. Envie as alterações para o repositório no GitHub:
git push -u origin main
